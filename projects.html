PROJECTS

In this project we are creating a basic ray tracer. Ray tracing models how light interactions work, and thus can help us create realistic renderings with features of real world light such as shadows, reflections, and diffusion. The goal of this project is to create a working ray tracer and to gain an intuitive understanding of how it works. This project is broken into seven steps which will also be how this report is organized. I loved seeing how each part of the ray tracer came together to create the final working implementation. Experimenting with each step and seeing what was accomplished and what has yet to be done was very satisfying.
